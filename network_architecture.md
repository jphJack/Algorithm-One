# VIBE双模态生物特征识别网络架构文档

## 1. 概述

VIBE（Vein and Palm Print Biometric Enhancement）网络是一个基于掌纹和掌静脉双模态的生物特征识别系统。该网络采用双层递进式混合专家（MoE）架构，通过多频域特征提取和跨模态融合机制，实现高精度的个人身份识别。

### 1.1 任务描述
- **目标**：基于掌纹和掌静脉图像判断个人身份
- **类别数**：290人
- **数据集**：每人5张训练图像和5张测试图像

### 1.2 数据规格
| 模态 | 图像尺寸 | 光源类型 | 文件夹 |
|------|----------|----------|--------|
| 掌纹 | 190×217 | 可见光 | print-train / print-test |
| 掌静脉 | 180×180 | 红外光 | vein-train / vein-test |

## 3. 模块详细设计

### 3.1 双支流骨干网络 (DualStreamBackbone)

骨干网络采用轻量化CNN架构，分别处理掌纹和掌静脉图像。两个支流具有独立参数，以适应不同模态的特征分布。

#### 3.1.1 单支流骨干网络结构

```
输入: [B, 1, H, W]
    │
    ▼
┌─────────────────────────┐
│ Stage 1: Conv(1→32, s=2)│  → 空间尺寸减半
│         Conv(32→32)     │
└───────────┬─────────────┘
            │
            ▼
┌─────────────────────────┐
│ Stage 2: Conv(32→64, s=2)│
│         Conv(64→64)      │
└───────────┬─────────────┘
            │
            ▼
┌─────────────────────────┐
│ Stage 3: Conv(64→128, s=2)│
│         Conv(128→128)    │
└───────────┬─────────────┘
            │
            ▼
┌─────────────────────────┐
│ Stage 4: Conv(128→256, s=2)│
│         Conv(256→256)    │
└───────────┬─────────────┘
            │
            ▼
┌─────────────────────────┐
│ Stage 5: Conv(256→256, s=2)│
│         Conv(256→256)    │
└───────────┬─────────────┘
            │
            ▼
输出: [B, 256, H/32, W/32]
```

每个卷积块包含：
- Conv2d (无bias)
- BatchNorm2d
- ReLU激活

### 3.2 MoE特征增强模块 (MoEEnhancement)

该模块为每种模态构建独立的专家网络池，包含三个并行专家：高频专家、中频专家和低频专家。门控网络根据输入特征动态生成权重，加权融合三个专家的输出。

#### 3.2.1 高频信息处理专家 (HighFreqExpert)

用于提取边缘、纹理等高频特征。

```
输入: [B, C, H, W]
    │
    ├──────────────────────────────────┐
    │                                  │
    ▼                                  ▼
┌─────────────────┐           ┌─────────────────┐
│ 拉普拉斯核卷积   │           │ 细节增强块1     │
│ (固定参数3×3)   │           │ (深度可分离卷积) │
│ [[0,1,0],       │           │ + 逐点卷积      │
│  [1,-4,1],      │           │ + BN + ReLU     │
│  [0,1,0]]       │           │ + 残差连接)     │
└────────┬────────┘           └────────┬────────┘
         │                             │
         │                             ▼
         │                    ┌─────────────────┐
         │                    │ 细节增强块2     │
         │                    │ (结构同上)      │
         │                    └────────┬────────┘
         │                             │
         └──────────┬──────────────────┘
                    │ (逐元素相加)
                    ▼
           ┌─────────────────┐
           │  1×1 Conv + BN  │
           └────────┬────────┘
                    │
                    ▼
输出: F_HF [B, C, H, W]
```

**细节增强块结构**：
```
输入 → 深度可分离卷积(3×3) → 逐点卷积(1×1) → BN → ReLU → (+输入) → 输出
```

#### 3.2.2 中频信息处理专家 (MidFreqExpert)

用于提取多尺度上下文信息和方向性纹理特征。

```
输入: [B, C, H, W]
    │
    ├───────────────────────────────────────┐
    │                                       │
    ▼                                       ▼
┌─────────────────────┐           ┌─────────────────────┐
│ 多尺度空洞卷积分支   │           │ 方向感知卷积分支     │
│                     │           │                     │
│ ┌─────┬─────┬─────┐│           │ ┌─────────┬───────┐ │
│ │1×1  │3×3  │3×3  ││           │ │ 1×7 Conv│7×1 Conv│ │
│ │Conv │d=2  │d=4  ││           │ │         │       │ │
│ │C/4  │C/4  │C/4  ││           │ │  C/2    │ C/2   │ │
│ └──┬──┴──┬──┴──┬──┘│           │ └────┬────┴───┬───┘ │
│    │     │     │   │           │      │Concat  │     │
│    └──┬──┴──┬──┘   │           │      ▼        │     │
│       │Concat      │           │ ┌─────────────┴───┐ │
│       ▼            │           │ │   1×1 Conv      │ │
│ ┌───────────┐      │           │ │   BN + ReLU     │ │
│ │1×1 Conv   │      │           │ └─────────────────┘ │
│ │BN + ReLU  │      │           └──────────┬──────────┘
│ └─────┬─────┘      │                      │
└───────┼────────────┘                      │
        │                                   │
        └──────────┬────────────────────────┘
                   │ (逐元素相加)
                   ▼
           ┌───────────────┐
           │  (+输入)      │ ← 残差连接
           └───────┬───────┘
                   ▼
           ┌───────────────┐
           │ 1×1 Conv + BN │
           └───────┬───────┘
                   ▼
输出: F_MF [B, C, H, W]
```

**参数说明**：
- d=2/4: 空洞卷积扩张率
- C/4: 各子分支输出通道数

#### 3.2.3 低频信息处理专家 (LowFreqExpert)

用于提取整体轮廓、光照分布等低频特征。

```
输入: [B, C, H, W]
    │
    ├───────────────────────────────────────┐
    │                                       │
    ▼                                       ▼
┌─────────────────────┐           ┌─────────────────────┐
│ 编码器-解码器分支    │           │ 低通滤波分支         │
│                     │           │                     │
│ ┌─────────────────┐ │           │ ┌─────────────────┐ │
│ │ 深度可分离卷积   │ │           │ │ 平均池化(5×5)   │ │
│ │ (7×7, s=1)      │ │           │ │ s=1, p=2        │ │
│ │ + 逐点卷积      │ │           │ └────────┬────────┘ │
│ │ + BN + ReLU     │ │           │          │          │
│ └────────┬────────┘ │           │          │          │
│          │          │           │          │          │
│          ▼          │           │          │          │
│ ┌─────────────────┐ │           │          │          │
│ │ MaxPool(2×2)    │ │           │          │          │
│ └────────┬────────┘ │           │          │          │
│          │          │           │          │          │
│          ▼          │           │          │          │
│ ┌─────────────────┐ │           │          │          │
│ │ 瓶颈层          │ │           │          │          │
│ │ Conv(3×3)       │ │           │          │          │
│ │ + BN + ReLU     │ │           │          │          │
│ └────────┬────────┘ │           │          │          │
│          │          │           │          │          │
│          ▼          │           │          │          │
│ ┌─────────────────┐ │           │          │          │
│ │ Upsample(×2)    │ │           │          │          │
│ │ + Concat(skip)  │ │           │          │          │
│ │ + Conv(1×1)     │ │           │          │          │
│ │ + BN + ReLU     │ │           │          │          │
│ └────────┬────────┘ │           │          │          │
│          │          │           │          │          │
│    F_dec │          │           │   F_lp   │          │
└──────────┼──────────┘           └──────────┼──────────┘
           │                                 │
           └──────────┬──────────────────────┘
                      │
                      ▼
           ┌─────────────────────┐
           │ F_fused = α×F_dec + │
           │          (1-α)×F_lp │ ← α为可学习参数(初始0.5)
           └──────────┬──────────┘
                      ▼
           ┌─────────────────────┐
           │   1×1 Conv + BN     │
           └──────────┬──────────┘
                      ▼
输出: F_LF [B, C, H, W]
```

#### 3.2.4 门控网络 (GateNetwork)

```
输入: [B, C, H, W]
    │
    ▼
┌─────────────────────┐
│ 全局平均池化         │
│ → [B, C]            │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ FC(C → C/4)         │
│ ReLU                │
│ FC(C/4 → 3)         │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ Softmax             │
│ → [w1, w2, w3]      │
└──────────┬──────────┘
           │
           ▼
输出: 专家权重 [B, 3]
```

#### 3.2.5 MoE特征增强模块整合

```
输出 = w1 × F_HF + w2 × F_MF + w3 × F_LF
```

### 3.3 MoE融合模块 (MoEFusion)

该模块构建跨模态交互专家池，包含三个异构专家：跨注意力融合专家、多尺度卷积融合专家和通道交互融合专家。

#### 3.3.1 跨注意力融合专家 (CrossAttentionExpert)

基于Transformer架构，利用双向交叉注意力机制捕捉双模态特征的全局语义关联。

```
输入: F_p [B, C, H, W], F_v [B, C, H, W]
    │
    ├──────────────────────────────────────────────────┐
    │                                                  │
    ▼                                                  ▼
┌───────────────────────────┐           ┌───────────────────────────┐
│ 掌纹→静脉注意力            │           │ 静脉→掌纹注意力            │
│                           │           │                           │
│ Q_p = Conv1×1(F_p)        │           │ Q_v = Conv1×1(F_v)        │
│ K_v = Conv1×1(F_v)        │           │ K_p = Conv1×1(F_p)        │
│ V_v = Conv1×1(F_v)        │           │ V_p = Conv1×1(F_p)        │
│                           │           │                           │
│ Attn = Softmax(Q_p @ K_v^T)│           │ Attn = Softmax(Q_v @ K_p^T)│
│ Cross_p2v = V_v @ Attn    │           │ Cross_v2p = V_p @ Attn    │
└─────────────┬─────────────┘           └─────────────┬─────────────┘
              │                                       │
              └──────────────────┬────────────────────┘
                                 │
                                 ▼
                    ┌───────────────────────────┐
                    │ Concat([F_p, F_v,         │
                    │        Cross_p2v,         │
                    │        Cross_v2p])        │
                    │ → [B, 4C, H, W]           │
                    └─────────────┬─────────────┘
                                  │
                                  ▼
                    ┌───────────────────────────┐
                    │ Conv1×1(4C → C) + BN      │
                    └─────────────┬─────────────┘
                                  │
                                  ▼
输出: [B, C, H, W]
```

#### 3.3.2 多尺度卷积融合专家 (MultiScaleConvExpert)

采用不同感受野的并行卷积核，聚焦局部微观特征的互补性。

```
输入: F_p [B, C, H, W], F_v [B, C, H, W]
    │
    ▼
┌───────────────────────────┐
│ Concat([F_p, F_v])        │
│ → [B, 2C, H, W]           │
└─────────────┬─────────────┘
              │
              ▼
┌───────────────────────────┐
│ Conv1×1(2C → C) + BN + ReLU│
└─────────────┬─────────────┘
              │
              ├──────────┬──────────┐
              │          │          │
              ▼          ▼          ▼
        ┌─────────┐┌─────────┐┌─────────┐
        │Conv 1×1 ││Conv 3×3 ││Conv 5×5 │
        └────┬────┘└────┬────┘└────┬────┘
             │          │          │
             └──────────┼──────────┘
                        │ (逐元素相加)
                        ▼
              ┌───────────────────┐
              │ Conv1×1 + BN      │
              └─────────┬─────────┘
                        │
                        ▼
输出: [B, C, H, W]
```

#### 3.3.3 通道交互融合专家 (ChannelInteractionExpert)

通过通道重校准机制，动态调整双模态特征通道的响应强度。

```
输入: F_p [B, C, H, W], F_v [B, C, H, W]
    │
    ├───────────────────────────────────────┐
    │                                       │
    ▼                                       ▼
┌─────────────────────┐           ┌─────────────────────┐
│ GlobalAvgPool(F_p)  │           │ GlobalAvgPool(F_v)  │
│ → [B, C]            │           │ → [B, C]            │
└──────────┬──────────┘           └──────────┬──────────┘
           │                                 │
           ▼                                 ▼
┌─────────────────────┐           ┌─────────────────────┐
│ MLP(Global_v)       │           │ MLP(Global_p)       │
│ FC(C→C/4)→ReLU→     │           │ FC(C→C/4)→ReLU→     │
│ FC(C/4→C)→Sigmoid   │           │ FC(C/4→C)→Sigmoid   │
│ → Weights_p         │           │ → Weights_v         │
└──────────┬──────────┘           └──────────┬──────────┘
           │                                 │
           ▼                                 ▼
┌─────────────────────┐           ┌─────────────────────┐
│ F_p × Weights_p     │           │ F_v × Weights_v     │
│ = F_p_calibrated    │           │ = F_v_calibrated    │
└──────────┬──────────┘           └──────────┬──────────┘
           │                                 │
           └─────────────────┬───────────────┘
                             │ (逐元素相加)
                             ▼
                   ┌───────────────────┐
                   │ Conv1×1 + BN      │
                   └─────────┬─────────┘
                             │
                             ▼
输出: [B, C, H, W]
```

#### 3.3.4 融合门控网络

```
输入: F_p [B, C, H, W], F_v [B, C, H, W]
    │
    ▼
┌───────────────────────────┐
│ GlobalAvgPool(F_p) → [B, C]│
│ GlobalAvgPool(F_v) → [B, C]│
│ Concat → [B, 2C]           │
└─────────────┬─────────────┘
              │
              ▼

└─────────────┬─────────────┘
              │
              ▼
输出: 专家权重 [B, 3]
```

```
输入: [B, C, H, W]
```

## 6. 关键设计特点

1. **双支流独立参数**：掌纹和掌静脉使用独立的骨干网络，适应不同模态的特征分布

2. **全专家协同策略**：摒弃传统的稀疏Top-k选择，采用全专家加权融合，最大化信息利用率

3. **多频域特征提取**：高频、中频、低频专家分别处理不同频率的特征，全面捕捉图像信息

4. **跨模态交互**：三种融合专家从不同维度挖掘掌纹与掌静脉的深层关联

5. **动态门控机制**：根据输入特征自适应调整专家权重，提升模型鲁棒性
